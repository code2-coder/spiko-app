# 🚀 Spiko – English Speaking Academy

Spiko is a modern English speaking academy platform designed to help students, job seekers, and working professionals improve their English communication skills and build real-world confidence.

---

## 🌟 Features

* 🎯 Modern & Clean Landing Page UI
* 📚 Course Showcase Section
* 💬 Student Testimonials
* 📞 WhatsApp Contact Integration
* 🧑‍🏫 Teacher / Hiring Section
* 📝 Lead Capture Form
* 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 Tailwind CSS
* 🧩 shadcn/ui
* 🎞️ Framer Motion

## 🚀 Getting Started

### 1. Clone Repository

```
git clone https://github.com/code2-coder/spiko-app.git
```

### 2. Install Dependencies

```
npm install
```

### 3. Run Development Server

```
npm run dev
```
